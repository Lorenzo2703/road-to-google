### CV Improvement
- [ ] Update CV to be 1 page (Google prefers concise CVs)
- [ ] Add quantifiable impact metrics to every bullet point (e.g., "improved uptime from 99.9% to 99.99%")
- [ ] Highlight experience with Python, C++, Java, Go, or JavaScript
- [ ] Add any infrastructure/distributed systems/Linux projects prominently
- [ ] Tailor the summary/objective section to SRE
- [ ] Remove irrelevant experience or skills
- [ ] Have 2-3 people review your CV
- [ ] Use action verbs: Designed, Built, Deployed, Optimized, Responded, Automated
- [ ] Ensure consistent formatting and zero typos
- [ ] Add a link to your GitHub profile

### Environment Setup
- [ ] Install Python 3.10+, Go, or preferred systems language
- [ ] Install Jupyter Notebook / JupyterLab
- [ ] Clone this repository
- [ ] Install all dependencies (`pip install -r requirements.txt`)
- [ ] Create a LeetCode account (Premium recommended)
- [ ] Create a Google Cloud account (free tier)
- [ ] Setup a Linux environment (VM or WSL) for systems practice
